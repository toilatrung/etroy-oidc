# eTroy OIDC - Handoff Template

## I. Purpose

This template standardizes end-of-session handoff so the next session can continue safely and quickly.

## II. When to Use This Template

Use this template at the end of any session that changes scope, decisions, documentation, implementation, or governance status.

## III. Required Handoff Fields

- Current phase
- Current sprint
- Current task
- Relevant source-of-truth documents
- What was completed
- What remains open
- Risks or blockers
- Next recommended action

- Ensure handoff statements do not override `docs/` authority.
